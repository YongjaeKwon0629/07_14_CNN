## 7. 참고 자료 (References)

CNN(Convolutional Neural Networks)은 매우 넓고 깊은 분야입니다. 아래는 CNN을 처음 접하는 학습자부터, 실무 프로젝트에 적용하려는 개발자까지 참고할 수 있는 신뢰도 높은 자료들을 정리한 목록입니다.

---

### 🎓 강의 및 튜토리얼

- 🔗 [CS231n: Convolutional Neural Networks for Visual Recognition (Stanford)](http://cs231n.stanford.edu/)
  - Stanford 대학의 대표적인 컴퓨터 비전 강의
  - 이론부터 실습까지 탄탄하게 구성됨
  - 💡 초심자에게도 매우 추천

- 🔗 [Deep Learning Specialization by Andrew Ng (Coursera)](https://www.coursera.org/specializations/deep-learning)
  - CNN뿐 아니라 딥러닝 전체의 큰 그림을 익힐 수 있음
  - 실제 코드 구현도 포함

- 🔗 [PyTorch 공식 튜토리얼 (60분만에 배우는 딥러닝)](https://pytorch.org/tutorials/beginner/deep_learning_60min_blitz.html)
  - CNN을 포함한 모델을 빠르게 구현할 수 있는 실용 예제 중심

---

### 📚 문서 및 책

- 📘 [Deep Learning with Python (François Chollet 저)](https://www.manning.com/books/deep-learning-with-python)
  - Keras 창시자가 쓴 명확하고 직관적인 딥러닝 입문서
  - CNN 예제와 시각화가 풍부

- 📘 [Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow (Aurélien Géron 저)](https://www.oreilly.com/library/view/hands-on-machine-learning/9781492032632/)
  - 실제 사례 위주의 실전서로, CNN 구현과 하이퍼파라미터 튜닝까지 포함

- 📘 [Dive into Deep Learning (D2L.ai)](https://d2l.ai/)
  - 온라인 무료 교재로 PyTorch, MXNet, TensorFlow 기반 예제 포함
  - 수학 설명 + 코드 구현 + 실습 병행

---

### 🔬 학술 논문 및 기술 문서

- 📄 [ImageNet Classification with Deep Convolutional Neural Networks (AlexNet, 2012)](https://papers.nips.cc/paper_files/paper/2012/file/c399862d3b9d6b76c8436e924a68c45b-Paper.pdf)
  - 현대 CNN 붐의 시작이 된 논문

- 📄 [Very Deep Convolutional Networks for Large-Scale Image Recognition (VGGNet, 2014)](https://arxiv.org/abs/1409.1556)
  - 구조가 단순하고 직관적인 VGG 모델 제안

- 📄 [Deep Residual Learning for Image Recognition (ResNet, 2015)](https://arxiv.org/abs/1512.03385)
  - 딥러닝에서 ‘Residual 연결’을 도입해 성능 향상

---

### 💻 GitHub 오픈소스 및 실습 코드

- 🔗 [PyTorch Examples (Official)](https://github.com/pytorch/examples)
  - 다양한 딥러닝 모델 예제가 공개되어 있음
  - CNN, RNN, GAN 등 실습 중심

- 🔗 [CNN Visualization Tool](https://github.com/yosinski/deep-visualization-tool)
  - CNN이 학습한 필터가 무엇을 인식하는지 시각화 가능
